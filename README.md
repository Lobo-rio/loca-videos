## Description

Study aimed at understanding the application of DDD at the code level, using some OO, SOLID, DDD and some patterns.
Pattern (Factory, InMemory Repository)

## Installation

```bash
$ git clone https://github.com/Lobo-rio/loca-videos.git
$ cd loca-videos
$ npm install
```

## Test

```bash
# unit tests
$ npm run test

# unit tests watch
$ npm run test:watch
```
